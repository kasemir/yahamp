ModelUtils.java

Fix for Eclipse bug 382184 that caused NPE when re-loading a saved model, requiring the command-line option -clearPersistedState to clean the workspace on each startup. Eric Moffatt posted this patch to the bug report.

  • In the IDE, open the Plug-ins view.
  • Import plugin org.eclipse.e4.ui.model.workbench as "Source Project"
  • In the imported source code, replace org.eclipse.e4.ui.model.internal.ModelUtils with the patched version.
  • 'Export' of the product from the IDE also seems to use the plugin sources from the workspace. No need to update the 'Target' platform.
